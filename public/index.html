A successful API call. Sentry records breadcrumbs for the request.

Unhandled JS Error

Throws an exception that is NOT caught — Sentry's global handler captures it automatically.

Handled JS Error

Wraps an operation in try/catch and calls Sentry.captureException() explicitly.

Server Crash

Calls /api/crash — the server throws, Sentry's Express middleware captures it.

Server Handled Error

Server catches the error internally and reports it via Sentry.captureException().

Slow Endpoint

Calls /api/slow which runs a traced span — visible in Sentry's performance view.

Custom Message

Sends a custom Sentry.captureMessage() from the frontend.
